QUAL É O PERSONAGEM?
Como funciona o jogo

  • Existe uma lista de 24 personagens que aparecem de forma aleatória
  • Seu objetivo é descobrir e acertar o nome de cada personagem
  • Caso não saiba o nome do personagem, aperte em Outro Personagem para pular o personagem
  • Caso saiba o nome do personagem, aperte em Verificar
*Personagens verificados corretamente, não se repetem
